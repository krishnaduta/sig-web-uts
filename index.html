Praktikum SIG Web pada minggu ke-1 hingga ke-7 lebih berfokus pada membangun sebuah web baik dari segi server (Server-Side Scripting) ataupun client (Client-Side Scripting). Untuk menunjang pembangunan web tersebut, maka diperkenalkanlah mahasiswa mengenai beberapa bahasa markup dan pemrograman, seperti HTML dan CSS untuk bahasa markupnya dan Javascript sebagai bahasa pemrogramannya.

Pada minggu ke-5 dan 6 diperkenalkanlah kepada leaflet js yang merupakan salah satu library dari javascript yang dapat membantu untuk membuat sebuah Web Map atau Peta berbasis web"
